LineDrawing
===========

Make [rtsc][1] simpler in Qt project

Features
--------

*	Cross-platform
*	So simpler

Keyboard shortcuts
------------------

* a: Apparent Ridges
* w: Suggestive Contours
* e: Edges
* l: Lights
and, ...
